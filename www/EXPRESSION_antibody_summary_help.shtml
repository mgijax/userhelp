Interpreting an Antibody Summary

This help document describes how to interpret an Antibody Summary.

If you reached this summary from a Gene Detail page, then the top of the page indicates the official gene symbol, linked to its Gene Detail page, and its name and MGI accession ID. If you reached this summary from a Reference Detail page, then the top of the page indicates the MGI reference ID, linked to the Reference Detail page, and the citation.

FIELD DESCRIPTION
AntibodyThe antibody name, linked to the Antibody Detail page; the MGI Accession ID of the antibody is also displayed (in parentheses).
GenesSymbol for each gene whose product is recognized by the antibody. This is the primary sort when the summary is accessed from the Reference Detail page.
TypeType of antibody; possible values are Monoclonal, Polyclonal, or Not Specified.
Host Organism in which the antibody was raised.
References The number of publications that use the antibody. This is the primary sort when the summary is accessed from a Gene Detail page.

Sequence Query Tips

This Help document answers the following questions about the Mouse Sequence Query Form:

  • How do I limit my query results?
  • Are there any tips for limiting query results?

How do I limit my query results?

To limit your query results, use as many fields as possible in your query. Override the default values with specific values you want to query. See the next question for specific suggestions in each search category. If it is not possible to limit your query and you wish to retrieve more than 5000 sequences, please Contact User Support and ask for a custom SQL query.

Are there any tips for limiting query results?

Yes. Experiment with some or all of the following.

Search Category Tip
Sequence Attributes
  • Change the Sequence Type default from ANY to a specific type.
  • Change the Sequence Provider default from ANY to one or more providers.
Sequence Source
  • Browse the strain list for a Strain you want to query.
  • Browse the tissue list for a Tissue you want to query.
  • Browse the library list for a Library you want to query.
  • Change the default operator from contains to = (is) when you want an exact match for Strain, Tissue, or Library.
Annotated Genes & Features
  • Enter a gene or marker by typing in a Marker Symbol/Name to find sequences associated to MGI genes or markers.
  • Change the default operator from contains to = (is) when you want an exact match for Marker Symbol/Name.
  • Change the marker Type default to a specific type.
Map Position
  • Select one or more Chromosomes.
  • Enter a cM Position.
  • Enter a Cytogenetic Band.
Gene Ontolotgy (GO) Classifications
  • Browse the GO classifications for a Gene Classification you want to query.
  • Change the default operator from contains to = (is) when you want an exact match for the Gene Classification.
Protein domains
  • Browse the domains for an InterPro Protein Domain you want to query.
  • Change the default operator from contains to = (is) when you want an exact match for the InterPro Protein Domain.
Expression
  • Select one or more Developmental Stage(s).
  • Browse the Anatomical Dictionary for an Anatomical Structures(s) you want to query.
  • Change the default operator from contains to = (is) when you want an exact match for the Anatomical Structures(s).
  • Select one or more Assay Type(s).
Phenotypes
  • Click Anatomical Systems Affected by Phenotypes and choose items from the list to add to your query.
  • Alternatively, browse the Full Mammalian Phenotype (MP) Ontology and/or the Human Disease (OMIM) Vocabulary for terms and definitions; type one or more terms into the Phenotype/Disease box.

What information is in a Protein Superfamily Detail report?

The Protein Superfamily Detail report displays protein classification sets obtained from the Protein Information Resource SuperFamily (PIRSF) site for a homeomorphic superfamily with mouse, human, and rat relationships from HomoloGene Class data. Each row contains links to the representative sequence (if present) of the mouse, human, and rat orthologs. You can select them by row and download them in FASTA format or forward them to MouseBLAST.

How do I interpret the Protein Superfamily Detail report resulting from my query?

The table is divided into three sections, horizontally, providing symbol, sequence, and chromosome information for the

  1. Mouse Superfamily Member
  2. MGI Human Ortholog
  3. MGI Rat Ortholog

If the human or rat ortholog is unknown, these fields (Symbol, Sequence, Chr) are blank.

See How do the buttons work? What does Invert do? for additional viewing options.

FIELD DESCRIPTION
Mouse PIRSF MembersThe official gene symbol for the mouse genetic marker.
Chr Chromosomal assignment of the mouse marker.
Homology Class The Homology Class ID from Homologene, linked to the MGI Vertebrate Homology Class page.
Mouse : Human : Rat Homology Class Shows the number of members in this Homology Class for mouse, human and rat.
Sequence
  1. The sequence identifier (as assigned by SwissProt) (i.e., Q61501).
  2. Name of and link to the sequence provider's detail page.
  3. In the Mouse Superfamily Member column, an additional link to the MGI Sequence Detail page (containing additional sequence information such as description, library, organism, strain, sex, age, tissue, cell line, GO terms, expression assays, orthologs, phenotypic alleles, references, and so on).
    Note: If the link (for mouse, human, or rat) does not appear, it means that as yet there is no sequence affiliated with this marker .
Chr Chromosomal assignment of the marker associated to the sequence.

How do I use a Protein Superfamily Detail report?

There are several ways to use the detail reports. You can:

  1. view sequence details for a single sequence by clicking either the MGI Sequence Detail or the provider name (e.g., SwissProt) link.
  2. download one or more sequences in FASTA format.
  3. forward one or more sequences to MouseBLAST.

How do I view details for a single sequence?

You can elect to view the sequence details in the provider's database or in MGI.

  1. Click the provider link (e.g., SwissProt).
    The provider's sequence record appears.
  2. Click MGI Sequence Detail.
    The detail record for that sequence as stored in MGI appears. See How do I interpret a Protein Superfamily Detail report resulting from my query?.

How do I download sequences in FASTA format?

To download sequences in FASTA format:

  1. Click to put a check mark in the Select column next to one or more sequences.
  2. Click the arrow in the box beneath the text For the selected sequences (above the table)
  3. Click download in FASTA format (this is the default).
  4. Click GO.

How do I forward sequences to MouseBLAST?

To forward sequences to MouseBLAST:

  1. Click to put a check mark in the Select column next one or more sequences.
  2. Click the arrow in the box beside the text For the selected sequences (above the table)
  3. Click forward to MouseBLAST.
  4. Click GO.

For additional information, see:

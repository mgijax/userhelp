Interpreting a Microarray Probeset Summary

The Microarray Probeset Summary report displays all probesets and arrays associated with a marker.

Field Description
Symbol
Name
ID		 Official symbol, name, and MGI accession identifier of the marker with the probeset/array.
Probesets The total number of probes associated with this marker (in MGI microarray annotation files) appears at the top of the column.
Probeset IDIdentifier of each probeset/array associated with this marker.
Microarray PlatformName of each probeset/array associated with this marker.
FTPName of and link to each assay report identified under Microarray Platform (reports are stored on the MGI FTP site).

Where can I find antibody records in the MGI databases?

Antibody records are associated with gene expression assays, antigens, genes, and references in the database. In MGI, these relationships are reflected in the links provided between these different records.

  • If an assay uses an antibody, the assay detail record provides a link to the antibody record.
  • If a gene is known to code for the antigen detected by an antibody, the detail record for the gene includes a link to the antibody in the molecular reagents portion of the marker detail record.
  • If a paper reports an antibody, the reference detail record includes a link to the antibody record.

You can access antibody records through these links, or, you can enter the unique MGI accession ID (e.g., MGI:1332271) in the Quick Search box located at the top of most MGI web pages.

How do I interpret an antibody query results summary?

When you select a link to an antibody, a summary description with the following fields appears:

Field Description
Antibody detailsMGI Accession ID of the antibody, linked to its detail record.
Name Antibody name.
Type Type of antibody (polyclonal, monoclonal, or Not specified).
Genes Symbols for genes that encode for the antigen detected by the antibody, linked to its marker detail record.
Reference J number (unique MGI reference accession ID) and short format citation for the reference in which this antibody is used, linked to the reference detail.

How do I interpret an antibody detail record?

FieldDescription
Name Antibody name
MGI Accession ID MGI Accession ID for the antibody.
Raised in Organism in which the antibody was raised.
Type Type of antibody (polyclonal, monoclonal), or Not specified.
Class Immunoglobulin class to which the antibody belongs (applies only to monoclonal antibodies).
Note Additional information about the antibody, if available.
Recognizes product in Western blotYes, No, or Not Specified.
Recognizes product in Immunoprecipitation Yes, No, or Not Specified.
Antigen Name of the protein or protein fragment against which the antibody was raised, linked to the antigen record.
Antigen recognition note Additional information regarding the specificity of the antibody, if available.
Reference Reference for a paper that uses the antibody. J number (unique MGI reference accession ID) linked to the reference detail, and short format citation.
Genes (Chromosome) Symbol for the gene that encodes for the antigen. The chromosome if known, appears in parentheses.
Aliases (Reference)Other names for the antibody, followed by the J number (unique MGI reference accession ID) of the reference the name is used in, linked to the reference detail.
Additional information Links to gene expression records of assays that use the antibody.

How do I interpret an antigen detail record?

Antigen descriptions are linked to antibody detail records. You can access antigen information by the antigen link in the antibody detail record, or you can use the Quick Search too to look for an antigen by its MGI Accession ID.

FieldDescription
Name Antigen name.
MGI Accession IDMGI accession ID for the antigen.
SpeciesSpecies from which the antigen was isolated.
StrainStrain of mouse from which the antigen was isolated, if specified.
SexSex of the animal from which the antigen was isolated, if specified.
AgeAge of the animal from which the antigen was isolated, if specified.
Tissue Tissue from which the antigen was isolated, if specified.
Tissue descriptionTreatment of the tissue from which the antigen was isolated, if applicable.
Cell LineCell line from which the antigen was isolated, if specified.
Region coveredRegion of the protein corresponding to the antigen.
Notes Additional information about the antigen, if available.
Modification DateDate record was last modified.
Antibodies Names of antibodies that recognize the antigen, linked to the antibody records.

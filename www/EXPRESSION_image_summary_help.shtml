Interpreting the Gene Expression Data Images Query by Gene -- Summary

How do I interpret this report?

When there are images of expression data for a given gene, the word Images, followed by a number, appears in the Expression section of the Gene Detail page. Clicking the number returns a summary page where you can access all that gene's expression images.

  • The symbol, name, and MGI accession ID of the gene whose expression images are displayed on this report appear at the top of the page.

  • The total number of expression images (panes) associated with expression data for this gene appears near the top of the page. Each figure contains at least one expression image (pane); some contain many. There is one thumbnail for each figure in the summary.

  • To the right of each thumbnail is the reference where the data was obtained, its figure label, and the type of expression study or studies illustrated in the figure, preceded by the appropriate pane labels (if given). Only expression studies of this gene are reported.

  • Click the thumbnail or figure label to access the image detail page. There you can view a full-sized image and find links to all assays that refer to this image.

  • Thumbnails are first sorted by the assay types they illustrate, then by year of publication and finally by reference (J) number. Figures showing only in situ assays are first, followed by figures showing both in situ and gel assays, and concluding with figures showing gel assays only.

  • Click Next or Previous to navigate the image set. Previous functions like a Back button. First returns you to the initial summary page; Last takes you to the last summary page.

  • To change the number of figures shown per page, use the drop-down box in the upper right. Choose between 10, 25, or 50 figures per page or All. If you have a slow computer connection, it is advisable to choose the smaller numbers.
